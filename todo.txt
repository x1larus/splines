---------------------------------------------------------------------
общий вид сплайна: S[i] = a[i] + b[i](x-x[i]) + c[i](x-x[i])^2 + d[i](x-x[i])^3
    где x[i] дано, а остальные 4 коэфа надо найти, 
    и! 0 <= i < n-1, то есть для 4 узлов 3 сплайна
---------------------------------------------------------------------
количество необходимых коэффициентов = количество уравнений = (n-1)*4
---------------------------------------------------------------------
Условия кусков сплайна:
    1. Куски должны проходить через свои узловые точки
    2. В стыках кусков должна обеспечиваться плавность:
        S'[i] = S'[i+1]
        S''[i] = S''[i+1] 0 <= i < n-2
    3. В начальной и конечной точках кривизна должна быть нулевая:
        S''[0] = 0 в точке x[0]
        S''[n-2] = 0 в точке x[n-1]!!!!!!!!!
---------------------------------------------------------------------
S'[i] = b[i] + 2c[i](x-x[i]) + 3d[i](x-x[i])^2
S''[i] = 2c[i] + 6d[i](x-x[i])
---------------------------------------------------------------------
мое представление коэффициентов в матрице и не только:
a[0] | b[0] | c[0] | d[0] | a[1] ...
---------------------------------------------------------------------
